## 2. الخطوط (Typography)

### الخط العربي الموصى به
- **للعناوين والنصوص**: Cairo، Tajawal، أو IBM Plex Sans Arabic.
- **الأوزان**:
  - عادي (Regular, 400): النصوص الثانوية، الأوصاف.
  - غليظ (Bold, 700): العناوين، الكلمات المفتاحية.
  - دقيق (Light, 300): التفاصيل الصغيرة تحت العناوين.

### الخط اللاتيني (للفرنسية والإنجليزية)
- **Inter** أو **Plus Jakarta Sans** للأرقام والكلمات اللاتينية.
- مزج Arabic + Latin يجب أن يكون متجانسًا في الحجم والارتفاع (baseline).

## 3. التخطيط (Layout)

### الحاوية القصوى (Max Width)
- Desktop: `max-width: 1200px; margin: 0 auto;`
- Tablet: `max-width: 100%; padding: 0 24px;`
- Mobile: `max-width: 100%; padding: 0 16px;`

### المسافات (Spacing Scale)
استخدم نظام مضاعفات 4px:
- 4px, 8px, 12px, 16px, 24px, 32px, 48px, 64px, 96px

### الزوايا (Border Radius)
- مستطيلات الاختيار (الأزرار، البطاقات): `border-radius: 8px`
- المستطيلات الصغيرة الذهبية للأرقام: `border-radius: 50%` (دائرية) أو `12px` (زوايا معدلة دائريًا حسب الوصف الأصلي)
- البطاقات الكبيرة: `border-radius: 12px`

### الفواصل الأفقية
- "خط أفقي رفيع يقسم الصفحة" → `border-top: 1px solid #E5E5E5` أو SVG line.
- يُستخدم لفصل الأقسام البصرية (مثلًا: انتقال من خلفية أزرق بحري إلى بيضاء).

## 4. دعم اللغتين: العربية (RTL) والفرنسية (LTR)

> هذا **اختبار حقيقي لمهارة المطور**. لا تعني فقط ترجمة الكلمات، بل عكس التصميم بالكامل.

### القاعدة العامة
```html
<html dir="rtl" lang="ar">  <!-- النسخة العربية -->
<html dir="ltr" lang="fr">  <!-- النسخة الفرنسية -->
```

### ما يجب أن ينعكس
1. **الرأس**: المستطيلات الخمسة تنتقل من اليمين إلى اليسار في النسخة العربية.
2. **الأيقونات والأرقام**: في الخطوات الأربع، المستطيل الذهبي الصغير ينتقل من الزاوية العليا اليمنى (في RTL) إلى الزاوية العليا اليسرى (في LTR).
3. **الأزرار**:
   - في RTL: "السابق" يمين، "التالي" يسار.
   - في LTR: "السابق" يسار، "التالي" يمين.
4. **الأسهم في شريط المراحل**: تنعكس اتجاهاتها.
5. **الهوامش (Margin/Padding)**: استخدم `margin-inline-start` و `margin-inline-end` بدل `margin-left`/`margin-right` ليتكيف تلقائيًا مع الاتجاه.
6. **الأيقونات الاتجاهية**: استخدم `transform: scaleX(-1)` للأيقونات التي تشير لجهة معينة (مثل السهم ←).

### تكوين Tailwind (إن استُخدم)
```js
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      colors: {
        navy: '#0A2540',
        gold: '#D4AF37',
      }
    }
  },
  // تفعيل دعم RTL
  important: true,
}
```

### مبدّل اللغة
```tsx
function LanguageToggle() {
  const { locale, toggleLocale } = useLocale();
  return (
    <button onClick={toggleLocale}>
      {locale === 'ar' ? 'FR' : 'ع'}
    </button>
  );
}
```

## 5. الهاتف أولاً (Mobile-First)

> 80% من مستخدمي المنصة في الجزائر سيفتحونها على هواتفهم. **لا تتنازل عن هذا المبدأ**.

### القواعد
1. **ابدأ بتصميم Mobile ثم انتقل إلى Desktop** (وليس العكس).
2. **المستطيلان "أنا بائع" و"أنا مشترٍ"** يجب أن يكونا بحجم إبهام اليد (min-height: 56px, min-width: 200px).
3. **النصوص قابلة للقراءة دون تكبير**: الحد الأدنى 14px للجوال.
4. **الأزرار قابلة للنقر بإبهام واحد**: min-height: 48px.
5. **الحقول في النماذج**: min-height: 48px، padding داخلي مريح.
6. **تجنب الـ hover-only interactions** — على الجوال لا يوجد hover.

### Breakpoints مقترحة
```css
/* Mobile-first */
.container { /* default: mobile */ padding: 0 16px; }

@media (min-width: 768px) {
  /* Tablet */
  .container { padding: 0 24px; }
}

@media (min-width: 1024px) {
  /* Desktop */
  .container { max-width: 1200px; margin: 0 auto; padding: 0 32px; }
}
```

## 6. المكوّنات البصرية المتكررة

### زر ذهبي أساسي
```tsx
<button className="
  bg-gold text-navy font-bold
  px-6 py-3 rounded-lg
  hover:bg-gold-dark transition
  disabled:opacity-50 disabled:cursor-not-allowed
  min-h-12
">
  التالي
</button>
```

### زر نصي (Text Button)
```tsx
<button className="text-navy underline hover:text-gold transition">
  إلغاء
</button>
```

### بطاقة بخلفية رمادية
```tsx
<div className="bg-gray-100 rounded-xl p-6 relative">
  <div className="absolute top-4 right-4 rtl:left-4 rtl:right-auto
                  bg-gold text-navy rounded-full w-12 h-12
                  flex items-center justify-center font-bold">
    01
  </div>
  <h3 className="font-bold text-lg mt-8">البائع/المؤجر يدخل عقاره</h3>
  <p className="text-sm text-gray-600 mt-2">يملأ نموذجًا ديناميكيًا...</p>
</div>
```

### تنبيه أمني 🔒
```tsx
<div className="flex items-center gap-2 text-sm text-navy bg-gold/10 p-3 rounded-lg">
  <span>🔒</span>
  <span>هذا الرقم سري ومشفّر بالكامل — لن يظهر لأي مشترٍ أو طرف ثالث.</span>
</div>
```

## قائمة فحص نظام التصميم

- [ ] الأزرق البحري #0A2540 مطبّق في الرأس والتذييل والـ Hero.
- [ ] الذهبي #D4AF37 مطبّق في كلمة "Match"، الأزرار الأساسية، الأيقونات المميزة.
- [ ] الأبيض خلفية للأقسام الفاتحة.
- [ ] الرمادي خلفية لبطاقات الخطوات الأربع وقسم الإحصائيات.
- [ ] الأحمر مقتصر على أيقونة "ممنوع" فقط.
- [ ] الأخضر لمؤشر المطابقة وعلامة الصح.
- [ ] الأصفر لتنبيه التقييد.
- [ ] خط عربي واضح (Cairo / Tajawal / IBM Plex Sans Arabic).
- [ ] خط لاتيني متناسق (Inter).
- [ ] أحجام خطوط متجاوبة (Mobile vs Desktop).
- [ ] max-width 1200px للحاوية في Desktop.
- [ ] Spacing scale 4px مضاعفات.
- [ ] Border radius 8px للأزرار، 12px للبطاقات.
- [ ] دعم RTL/LTR كامل مع انعكاس التخطيط.
- [ ] Mobile-First: إبهام اليد، نصوص ≥14px، أزرار ≥48px.
- [ ] مكوّنات قابلة لإعادة الاستخدام (Button, Card, Alert).
- [ ] الأيقونات موحدة عبر المنصة.
